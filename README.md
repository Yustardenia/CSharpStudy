# CSharpStudy

一个用来记录 LeetCode 刷题、思路复盘和小型可视化练习页面的个人学习仓库。

## 内容概览

- `Program.cs`
  当前正在手敲和练习的题目代码。

- `Notes/`
  每道题的题解、错误记录、复盘笔记。

- `Demos/`
  一些前端可视化页面，用来帮助理解算法过程。

- `InterviewPrep/`
  面试题库与练习页面相关文件。

可以直接用浏览器打开这些文件：

- `Demos/median.html`
- `Demos/regex-dp.html`
- `InterviewPrep/index.html`

## 使用方式

如果只是看题解和笔记，直接打开 `Notes/` 即可。  
如果要运行当前 C# 代码，可以在仓库根目录执行：

```powershell
dotnet run
```

## 仓库目标

这个仓库主要不是追求“堆题量”，而是：

- 记录每题为什么会错
- 把思路变成可以重复复习的笔记
- 用更直观的页面理解难点题目的过程
